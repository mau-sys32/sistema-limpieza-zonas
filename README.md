# Sistema de Gestión y Control de Limpieza por Zonas

Proyecto desarrollado como parte de prácticas profesionales.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- Node.js y express (API REST)
- Firebase (Base de datos en la nube)

## Funcionalidades actuales
- Gestión de zonas
- Asignación de tareas
- Control de personal
- Historial de actividades
